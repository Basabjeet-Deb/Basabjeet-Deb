# 💫 About Me:

### AI/ML, IOT builder

I'm a passionate Data Scientist with hands-on experience in IoT development and AI/ML leadership. Currently pursuing a B.Sc. in Data Science while actively contributing to the tech ecosystem through internships and leadership roles.

My professional journey includes 3 months as an IoT Intern at GroveUs Informatics, where I gained comprehensive experience in both software and hardware integration. Currently, I serve as AI/ML Lead at E-Cell Inspiria Knowledge Campus, driving innovation and mentoring fellow students in cutting-edge technologies.

My expertise spans statistical methods, Python programming, IoT systems, and creating compelling data visualizations using Tableau and Power BI. I specialize in developing end-to-end solutions from hardware integration to advanced machine learning models.
